## 🚀 How to Run Locally

### 1️⃣ Clone the repo

git clone https://github.com/chaitragorla/real-time-social-media-app.git

cd real-time-social-media-app

### 2️⃣ Run Backend

cd server
npm install
npm run dev

👉 Runs on: http://localhost:3000

### 3️⃣ Run Frontend

cd client
npm install
ng serve --open

👉 Runs on: http://localhost:4200
